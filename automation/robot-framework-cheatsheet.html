🔄 Control Flow

# IF / ELSE IF / ELSE
IF    '${env}' == 'prod'
    Log    Production environment
ELSE IF    '${env}' == 'staging'
    Log    Staging environment
ELSE
    Log    Dev environment
END

# FOR loop
FOR    ${item}    IN    @{ITEMS}
    Log    ${item}
END

# FOR with range
FOR    ${i}    IN RANGE    1    10
    Log    Iteration ${i}
END

# WHILE loop (RF 5+)
WHILE    ${count} < 10
    ${count}    Evaluate    ${count} + 1
END

# TRY / EXCEPT
TRY
    Click Element    id=mayNotExist
EXCEPT    ElementNotFound*    type=GLOB
    Log    Element not found, continuing
END

💡 Pro Tips

🎯 Use Page Object Model

Create separate .robot resource files for each page with its own keywords. Keeps tests clean and maintainable.

⏱️ Avoid Hard Sleeps

Never use Sleep. Always use Wait Until Element Is Visible or Wait Until Page Contains for dynamic content.

🏷️ Tag Everything

Use tags like smoke, regression, sanity, login to run subsets of tests. Makes CI/CD pipeline management easy.

📸 Screenshots on Failure

Add "Run Keyword If Test Failed Take Screenshot" in Test Teardown. Saves hours of debugging.

🔑 Externalize Test Data

Keep test data in CSV or Excel files. Use DataDriver library for data-driven tests. Never hardcode credentials.

🚀 Parallel Execution

Use pabot (Parallel Robot) for parallel test execution: pip install robotframework-pabot then pabot tests/
