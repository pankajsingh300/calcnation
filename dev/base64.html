What is Base64?

Base64 is an encoding scheme that converts binary data into ASCII text. It's widely used to embed images in HTML/CSS, store data in JSON, transmit data in URLs, and encode email attachments.

A Base64-encoded string is about 33% larger than the original data. Use it when you need to represent binary data as text, not as a compression method.

Frequently Asked Questions

Is Base64 the same as encryption? +
No. Base64 is encoding, not encryption. Anyone can decode a Base64 string — it provides no security. Don't use it to hide sensitive data.
What files can I encode to Base64? +
Any file — images, PDFs, fonts, audio. The output is a data URI you can embed directly in HTML or CSS.
