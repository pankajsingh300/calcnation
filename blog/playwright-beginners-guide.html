Playwright for Beginners: Your First Automated Test

📅 Mar 8, 2026 · 🔖 Automation · ⏱️ 8 min read

If you're new to test automation, Playwright is one of the easiest and most powerful tools to get started. In this guide, we'll write your first automated test in under 10 minutes.

What is Playwright?

Playwright is a free, open-source automation library for testing web applications across Chrome, Firefox, and Safari. It's maintained by Microsoft and loved by QA engineers worldwide.

Why Playwright?

  • Fast: Runs tests in seconds, even across multiple browsers
  • Reliable: Built-in waiting mechanisms avoid flaky tests
  • Easy to learn: Simple syntax, great documentation
  • Free: No licensing costs, open source forever

Step 1: Install Playwright

Open your terminal and run:

npm install -D playwright
npx playwright install

Step 2: Write Your First Test

Create a file called test.js:

const { test, expect } = require('@playwright/test');

test('Google search works', async ({ page }) => {
  // Navigate to Google
  await page.goto('https://www.google.com');
  
  // Type in search box
  await page.fill('input[name="q"]', 'Playwright automation');
  
  // Press Enter
  await page.press('input[name="q"]', 'Enter');
  
  // Wait for results and verify
  await page.waitForSelector('#search');
  await expect(page).toHaveTitle(/Playwright/);
});

Step 3: Run the Test

npx playwright test test.js

Congratulations! Your first test is running. 🎉

Key Playwright Methods to Know

  • page.goto(url) — Navigate to a URL
  • page.fill(selector, text) — Type into input fields
  • page.click(selector) — Click buttons
  • page.waitForSelector(selector) — Wait for element to appear
  • expect(page).toHaveTitle(...) — Assert page title

Common Selectors

// CSS Selector
page.click('.button-class')

// ID Selector
page.click('#submit-button')

// XPath
page.click('//button[text()="Submit"]')

// Attribute selector
page.fill('input[type="email"]', 'test@example.com')

Debugging Tests

Use the Playwright Inspector to debug tests interactively:

PWDEBUG=1 npx playwright test test.js

This opens a browser with a step-by-step debugger so you can see exactly what your test is doing.
